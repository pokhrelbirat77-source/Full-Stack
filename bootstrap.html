Framework and Components

Bootstrap

Bootstrap is a CSS framework that provides ready-made layouts and components. It is useful for building neat responsive pages faster.

Why Use Bootstrap?

Bootstrap saves time by giving tested classes for grids, buttons, forms, navbars, alerts, and spacing. It also helps beginners make pages responsive without writing everything from zero.

  • Consistent layout
  • Mobile-first grid
  • Reusable components
  • Good documentation

Grid System

The grid is based on 12 columns. A layout can be one column on mobile and multiple columns on desktop.
