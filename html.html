Web Page Structure

HTML

HTML is the foundation of every web page. It gives meaning to text, links, images, tables, forms, and page sections.

<header><nav><main><section><form><footer>

Basic Document Structure

An HTML document starts with a doctype, then an html element. The head contains information for the browser, and the body contains what users see.

<!DOCTYPE html>
<html lang="en">
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Welcome</h1>
</body>
</html>

Semantic Tags

Semantic tags are more than decoration. They tell the browser which part is the heading area, navigation, main content, side content, and footer.

header
nav
main
section
article
aside
footer
<main>
  <section>About this topic</section>
  <article>Main explanation</article>
</main>

Lists and Links

Lists group related items. Links connect pages and make navigation possible.

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<a href="css.html">Go to CSS</a>

Images and Accessibility

Images are displayed with the <img> tag. The alt text should explain the image for screen readers and for cases where the image does not load.

<img src="images/student-work.svg"
     alt="Student working on a laptop at a desk">
