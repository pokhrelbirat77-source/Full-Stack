# ePortfolio - Birat Pokhrel
## First Year Computer System Engineering Student

This is my ePortfolio website for my first year web development assignment. The site uses Tailwind CSS as the styling library, while the learning pages explain Full Stack Development, HTML, CSS, Bootstrap, and JavaScript.

## Files

- `index.html` - welcoming starting page
- `about.html` - about me, goals, tools, and quick details
- `contact.html` - contact information and validated contact form
- `fullstack.html` - full stack development explanation, request flow, and CRUD examples
- `html.html` - HTML notes, semantic layout, forms, tables, images, and accessibility
- `css.html` - CSS selectors, box model, colours, flexbox, and responsive design
- `bootstrap.html` - Bootstrap notes, grid, components, and class examples
- `javascript.html` - JavaScript notes and interactive examples
- `style.css` - simplified custom styling used by all pages
- `script.js` - JavaScript used by the interactive examples
- `images/student-work.svg` - image example for the HTML accessibility page

## Final Notes

- Tailwind CDN is used instead of Bootstrap CDN.
- Bootstrap is still included as a learning topic page.
- The navbar is custom CSS/JavaScript because Bootstrap JS is no longer used.
- The CSS file is simplified and grouped by purpose.

## How to Run

Open `index.html` in a web browser. The topic pages can be opened from the navigation bar. No server is required because this is a static website.
