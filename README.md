# ePortfolio - Birat Pokhrel
## First Year Computing Student

This is my ePortfolio website for my first year web development assignment. It explains the basic ideas I have learned in full stack development, HTML, CSS, Bootstrap, and JavaScript.

## Files

- `index.html` - introduction, about me, and contact
- `fullstack.html` - full stack development explanation, request flow, and CRUD examples
- `html.html` - HTML notes, semantic layout, forms, tables, and accessibility
- `css.html` - CSS selectors, box model, colours, flexbox, and responsive design
- `bootstrap.html` - Bootstrap grid, components, buttons, alerts, and forms
- `javascript.html` - JavaScript notes and interactive examples
- `style.css` - custom styling used by all pages
- `script.js` - JavaScript used by the interactive examples
- `README.md` - project notes

## Pages Included

## JavaScript Features

- Typing text in the hero section
- Active navigation link while scrolling
- Active topic link on separate pages
- Flexbox alignment demo
- Calculator
- Character counter
- Random quote generator
- Form validation

## How to Run

Open `index.html` in a web browser. The topic pages can be opened from the navigation bar. No server is required because this is a static website.
