JavaScript

JavaScript adds behaviour to web pages. It can respond to events, validate forms, update content, and create interactive tools.

clickinputcalculateupdate DOM

Variables, Functions, and Events

JavaScript stores values in variables, groups reusable code in functions, and runs code when events happen.

var name = 'Birat';

function greet() {
  alert('Hello ' + name);
}

button.addEventListener('click', greet);

DOM Manipulation

The DOM is the browser's version of the page. JavaScript can find elements and change their text, style, or classes.

var title = document.getElementById('mainTitle');
title.textContent = 'Updated heading';
title.classList.add('active');
